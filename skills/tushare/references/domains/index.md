# Index Domain Interfaces

## 指数专题

- `rt_idx_min`: 指数实时分钟 - 获取交易所指数实时分钟数据，包括1\~60min
- `idx_mins`: 指数历史分钟 - 获取交易所指数分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK和 http Restful API两种方式
- `daily_info`: 沪深市场每日交易统计 - 获取交易所股票交易统计，包括各板块明细
- `index_global`: 国际主要指数 - 获取国际主要指数日线行情
- `index_classify`: 申万行业分类 - 获取申万行业分类，可以获取申万2014年版本（28个一级分类，104个二级分类，227个三级分类）和2021年本版（31个一级分类，134个二级分类，346个三级分类）列表信息
- `sz_daily_info`: 深圳市场每日交易情况 - 获取深圳市场每日交易概况
- `index_dailybasic`: 大盘指数每日指标 - 目前只提供上证综指，深证成指，上证50，中证500，中小板指，创业板指的每日指标数据
- `rt_idx_k`: 指数实时日线 - 获取交易所指数实时日线行情，支持按代码或代码通配符一次性提取全部交易所指数实时日k线行情
- `index_weight`: 指数成分和权重 - 获取各类指数成分和权重，**月度数据** ，建议输入参数里开始日期和结束日分别输入当月第一天和最后一天的日期。
- `ci_index_member`: 中信行业成分 - 按三级分类提取中信行业成分，可提供某个分类的所有成分，也可按股票代码提取所属分类，参数灵活
- `idx_factor_pro`: 指数技术面因子(专业版) - 获取指数每日技术面因子数据，用于跟踪指数当前走势情况，数据由Tushare社区自产，覆盖全历史；输出参数_bfq表示不复权描述中说明了因子的默认传参，如需要特殊参数或者更多因子可以联系管理员评估，指数包括大盘指数 申万行业指数 中信指数
- `index_member_all`: 申万行业成分(分级) - 按三级分类提取申万行业成分，可提供某个分类的所有成分，也可按股票代码提取所属分类，参数灵活
- `rt_sw_k`: 申万实时行情 - 获取申万行业指数的最新截面数据
- `index_monthly`: 指数月线行情 - 获取指数月线行情,每月更新一次
- `ci_daily`: 中信行业指数日行情 - 获取中信行业指数日线行情
- `index_weekly`: 指数周线行情 - 获取指数周线行情
- `sw_daily`: 申万行业指数日行情 - 获取申万行业日线行情（默认是申万2021版行情）
- `index_basic`: 指数基本信息 - 获取指数基础信息。

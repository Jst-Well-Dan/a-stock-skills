# Global Domain Interfaces

## 外汇数据

- `fx_obasic`: 外汇基础信息(海外) - 获取海外外汇基础信息，目前只有FXCM交易商的数据
- `fx_daily`: 外汇日线行情 - 获取外汇日线行情

## 港股数据

- `hk_cashflow`: 港股现金流量表 - 获取港股上市公司现金流量表数据
- `hk_balancesheet`: 港股资产负债表 - 获取港股上市公司资产负债表
- `hk_income`: 港股利润表 - 获取港股上市公司财务利润表数据
- `hk_fina_indicator`: 港股财务指标数据 - 获取港股上市公司财务指标数据，为避免服务器压力，现阶段每次请求最多返回200条记录，可通过设置日期多次请求获取更多数据。
- `hk_adjfactor`: 港股复权因子 - 获取港股每日复权因子数据，每天滚动刷新
- `hk_daily_adj`: 港股复权行情 - 获取港股复权行情，提供股票股本、市值和成交及换手多个数据指标
- `rt_hk_k`: 港股实时日线 - 获取港股实时日k线行情，支持按股票代码及股票代码通配符一次性提取全部股票实时日k线行情
- `hk_basic`: 港股基础信息 - 获取港股列表信息
- `hk_daily`: 港股日线行情 - 获取港股每日增量和历史行情，每日18点左右更新当日数据
- `hk_mins`: 港股分钟行情 - 港股分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK和 http Restful API两种方式
- `hk_tradecal`: 港股交易日历 - 获取交易日历

## 美股数据

- `us_fina_indicator`: 美股财务指标数据 - 获取美股上市公司财务指标数据，目前只覆盖主要美股和中概股。为避免服务器压力，现阶段每次请求最多返回200条记录，可通过设置日期多次请求获取更多数据。
- `us_adjfactor`: 美股复权因子 - 获取美股每日复权因子数据，在每天美股收盘后滚动刷新
- `us_cashflow`: 美股现金流量表 - 获取美股上市公司现金流量表数据（目前只覆盖主要美股和中概股）
- `us_balancesheet`: 美股资产负债表 - 获取美股上市公司资产负债表（目前只覆盖主要美股和中概股）
- `us_income`: 美股利润表 - 获取美股上市公司财务利润表数据（目前只覆盖主要美股和中概股）
- `us_daily`: 美股日线行情 - 获取美股行情（未复权），包括全部股票全历史行情，以及重要的市场和估值指标
- `us_tradecal`: 美股交易日历 - 获取美股交易日历信息
- `us_basic`: 美股基础信息 - 获取美股列表信息
- `us_daily_adj`: 美股复权行情 - 获取美股复权行情，支持美股全市场股票，提供股本、市值、复权因子和成交信息等多个数据指标

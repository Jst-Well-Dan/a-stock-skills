# 巨潮资讯财务报告下载 (cninfo-report)

从巨潮资讯网（cninfo.com.cn）批量下载股票财务报告 PDF，支持沪深京 A 股和港股市场。

## 触发时机

用户提出以下需求时激活本 skill：
- "帮我下载 [股票代码] 的年报"
- "下载 [公司名/股票代码] 最近几年的财务报告"
- "批量下载 [股票代码] 的季报/半年报"
- "从巨潮下载 [股票代码] 的报告"
- 含有"财报下载"、"年报下载"、"cninfo"等关键词

## 脚本信息

- **脚本路径**：`skills/cninfo-report/scripts/cninfo-report.py`
- **输出目录**：运行时在**工作目录**下自动创建 `财务报告下载/` 子目录

## 执行流程

### Step 1：收集参数

向用户确认以下参数（可有默认值）：

| 参数 | 说明 | 默认值 |
|---|---|---|
| 股票代码 | A股6位数字 / 港股5位数字 | 必填 |
| 市场 | `沪深京` 或 `港股` | 沪深京 |
| 报告类型 | `年报` 或 `all`（年报+半年报+季报） | 年报 |
| 年数 | 最近N年，范围1-20 | 5 |

**代码格式说明：**
- 沪深京：6位数字，如 `000001`、`600036`、`000858`
- 港股：5位数字，如 `00700`（腾讯）、`01299`（友邦）

### Step 2：运行脚本

```bash
cd ".claude/skills/cninfo-report/scripts"
python cninfo-report.py <股票代码> [--market <市场>] [--type <报告类型>] [--years <年数>]
```

**典型命令示例：**

```bash
# 下载贵州茅台最近5年年报
python cninfo-report.py 600519

# 下载腾讯控股（港股）最近3年年报
python cninfo-report.py 00700 --market 港股 --years 3

# 下载平安银行最近10年所有报告（年报+半年报+季报）
python cninfo-report.py 000001 --years 10 --type all

# 下载招商银行最近3年年报
python cninfo-report.py 600036 --market 沪深京 --years 3
```

### Step 3：反馈结果

脚本运行结束后，告知用户：
1. 成功下载数量 / 总数量
2. 文件保存路径（`财务报告下载/<股票代码>_<简称>/`）
3. 如有失败，简述可能原因

## 依赖检查

```bash
pip install requests pandas
```

（`requests` 和 `pandas` 为标准库扩展，通常已安装）

## 故障排除

| 问题 | 解决方法 |
|---|---|
| `ModuleNotFoundError: requests` | `pip install requests` |
| `ModuleNotFoundError: pandas` | `pip install pandas` |
| 股票代码未找到 | 确认代码格式（A股6位 / 港股5位），且已在巨潮上市 |
| 下载的PDF过小（<1KB） | 链接失效，可能是巨潮服务器临时限制，稍后重试 |
| 网络超时 | 检查网络连接；脚本有3次自动重试 |
| 编码错误（Windows） | 脚本已内置 UTF-8 控制台修复，若仍有问题用 `chcp 65001` |

## 输出说明

**默认输出目录**：运行脚本时的工作目录下的 `财务报告下载/` 文件夹

```
<工作目录>/
└── 财务报告下载/
    └── <股票代码>_<简称>_年度报告/   ← 年报下载到此文件夹
        ├── 600519_贵州茅台_20240330_贵州茅台2023年年度报告.pdf
        └── ...
```

**文件夹命名规则**：`{股票代码}_{简称}_年度报告`

**文件命名规则**：`{股票代码}_{简称}_{日期}_{公告标题}.pdf`

示例：`600519_贵州茅台_20240330_贵州茅台2023年年度报告.pdf`

已存在的文件会自动跳过（断点续传）。
